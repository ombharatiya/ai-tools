# Image generation

Tools that create images from text prompts. They differ a lot in look, control, cost, and whether you can run them yourself.

**Last verified:** 2026-06-05

## Quick pick

- **You want the best looking results with the least effort:** [Midjourney](midjourney.md).
- **You are a developer, or you need accurate text inside images:** [OpenAI image generation](dall-e.md).
- **You want to run it locally, privately, and free, and you have a GPU:** [Stable Diffusion](stable-diffusion.md).
- **You need an API:** [OpenAI image generation](dall-e.md) or [Stable Diffusion](stable-diffusion.md). Midjourney has no official API.

## The honest summary

These three win for different reasons. Midjourney gives the most polished default look with the least prompt effort, which is why artists and marketers love it, but it has no official API, no free tier, and weak privacy. OpenAI image generation is the developer's choice: a clean official API, the best text rendering inside images, and strong editing, at the cost of token based pricing that is hard to predict and provenance watermarks on every output. Stable Diffusion is the open option you can run on your own machine for free, with the largest customization ecosystem and the best privacy, but it needs a capable GPU and a willingness to set things up.

One point that applies to all three: under current US copyright decisions, purely AI-generated images may not be registrable for copyright, regardless of what a tool's terms grant you. Only your human authored edits and arrangement are protectable. Confirm the legal position for your use before you rely on it.
